## Guidelines

- Use a dark terminal theme consistent with Tokyo Night colors
- Terminal font: Use a monospace font (e.g., JetBrains Mono, Fira Code)
- Ensure text is readable at standard documentation page width
- Prefer PNG format for screenshots, SVG for diagrams
- Keep file sizes reasonable (compress PNGs with tools like `pngquant`)
